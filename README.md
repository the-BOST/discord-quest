# discord-quest

--


Vous avez donc deux options, vous pouvez toujours ouvrir l'inspecteur web dans le navigateur. Ou vous pouvez le réactiver *en* définissant "DANGEROUS_ENABLE_DEVTOOLS_ONLY_ENABLE_IF_YOU_KNOW_WHAT_YOURE_DOING": true dans %appdata%/discord/settings.json.

Je vais cependant nuancer cela avec une clause de non-responsabilité importante :
- Ne collez pas dans la console ce que d'autres personnes vous envoient. De telles choses peuvent compromettre votre compte, et même votre ordinateur.
- Ne partagez pas votre écran ni n'envoyez de captures d'écran de l'inspecteur web, en particulier l'onglet *requêtes*. Des informations sensibles telles que les informations d'identification de votre compte peuvent être à l'écran et permettre à quelqu'un de compromettre votre compte.
